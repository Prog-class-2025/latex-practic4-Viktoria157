# Практична робота з LaTeX

## Тема:
Базові можливості LaTeX

## Що зроблено:
- створено документ - article
- додано структуру
- реалізовано зміст - \tableofcontents
- використано форматування - \textbf{відповідальний}, \textit{111 Математика}
- використаний маркований список - \begin{itemize}

## Висновок:
LaTeX дозволяє автоматично оформлювати документи.
